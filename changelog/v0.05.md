# v0.05 版本变更说明

> 发布日期：2026-06-05　|　对应分支/标签：main　|　负责人：henry + AI（技术总监）
> 主题：**前端视觉重构（浅色系设计）**

## 一、本次变更

### 修改 / 优化（refactor）
- 建立**浅色系设计令牌**（`src/style.css`）：自定义 Element Plus 主色为靛蓝 `#4f6ef7`、统一圆角/阴影/间距/字体、精致滚动条。
- **侧边栏由深色改为浅色**：白底 + 品牌标识 + 圆角高亮的菜单项（原 `#001529` 深色被替换 → 这是本次主因）。
- **顶栏**：显示当前页标题 + 头像下拉（退出登录）。
- **登录页重做**：浅色渐变背景、品牌卡片、图标输入框、渐变按钮。
- **工作台重做**：渐变欢迎横幅 + 4 张统计卡片 + “快速开始”步骤条。
- 数据源/任务/执行记录占位页统一为带页头的卡片风格。

### 新增（feat）
- `src/components/PageContainer.vue`：统一页头（标题 + 副标题 + 操作区）的可复用容器。
- 路由新增 `meta.title`，供顶栏显示当前页名。

### 删除
- 无代码删除（深色侧边栏为样式替换，非删除功能）。

## 二、验证（三维度审核）

- **维度一·功能正确性**：`npm run build`（含 vue-tsc 类型检查）通过、`eslint` 通过、`prettier` 已格式化。
- **维度二·健壮性/扩展性**：设计令牌集中在 `style.css`，配色/圆角可一处调整全局生效；`PageContainer` 复用降低重复。
- **维度三·关联影响**：纯前端视觉层改动，路由/登录态/接口调用逻辑未变；后端仅 health 版本号 +1，不影响功能。
