# v0.06 版本变更说明

> 发布日期：2026-06-05　|　分支：**dev**（重构基线大改版 · 阶段 A 地基）　|　负责人：henry + AI
> 主题：**后端数据模型 + 权限可见性判定**

## 一、本次变更

### 新增（feat）
- 新建六个业务 app（模型层）：
  - `accounts`：**自定义 User**（`is_assistant_admin` 辅助管理员 / `is_boss` 老板 + 内置 `is_superuser` 超管）；`AUTH_USER_MODEL = accounts.User`。
  - `catalog`：`Department` / `Category`（部门 → 分类）。
  - `datasource`：`DataSource`（密码 Fernet 加密，`password` 属性自动加解密）。
  - `dataset`：`Dataset`（SQL / 归属分类 / 数据源 / 定时规则）。
  - `execution`：`Execution`（状态 / 行数 / 文件路径 / 最新标志）。
  - `permission`：`DepartmentMembership`（部门内角色 总监/主管/成员 + 看本部门全部）、`Grant`（个人或角色组 → 分类或数据集）。
- **权限唯一出口** `permission/services.py::can_view_dataset`：默认拒绝，覆盖 超管/辅助管理员/老板 → 总监/主管 → 成员(看全部/授权) 全链路。
- 单元测试 `tests/test_permission.py`（11 例，覆盖各角色与授权路径）；引入 `pytest-django`。

### 修改 / 优化
- `config/settings.py`：注册六个 app + `AUTH_USER_MODEL`。
- `pyproject.toml`：pytest 增加 `DJANGO_SETTINGS_MODULE`。

### 数据库
- 按工程约定**清空 SQLite + 重建迁移**（无真实数据）；各 app 生成 `0001_initial`；重建超级管理员 admin。

### 删除
- 无。
