# v0.09 版本变更说明

> 发布日期：2026-06-05　|　分支：**dev**（重构基线大改版 · 阶段 B 功能）　|　负责人：henry + AI
> 主题：**数据集 + SQL→Excel 手动运行 + 管理端下载（🎯 核心闭环）**

## 一、本次变更

### 新增（feat）
- **核心取数闭环**（写 SQL → 运行 → 出 Excel → 下载）：
  - `core/oracle_client.py`：新增 `preview_query`（只取前 N 行、用完即关）。
  - `apps/dataset/services.py`：`preview_dataset` + `run_dataset`（连 Oracle 流式取数 → openpyxl 生成 Excel → 写 Execution、切换 is_latest；失败落 error_msg 不抛 500）。
  - `apps/dataset`：序列化（**保存即校验只读 SQL**）+ ViewSet（CRUD + `preview` + `run`）。
  - `apps/execution`：序列化 + 只读 ViewSet（列表 ?dataset 过滤 + `download`，FileResponse 流式、中文名 RFC5987）。
  - 路由注册 `/api/datasets/`、`/api/executions/`。
- 前端：
  - `api/dataset.ts`、`api/execution.ts`（含带 Token 的 blob 下载）。
  - `views/admin/DatasetView.vue`：表格 + 新建/编辑（选数据源 + SQL 文本域）+ 运行 + **预览前 50 行** + 下载最新 + 删除。
  - `views/admin/ExecutionView.vue`：执行记录列表 + 下载。
- 单元测试 `tests/test_dataset_api.py`（8 例）：SQL 校验拒写操作、创建设 owner、预览(mock)、运行生成 Excel(mock)、仅最新版本、失败记录、下载、非管理员 403。

### 修改 / 优化
- `Dataset.category` 改为**可空**（未归类草稿；目录归类在 v0.10），并**移除** `unique_together(category,name)`（可空后该约束不再合适）→ 迁移 0002 / 0003。
- `health` 版本号 → v0.09。

### 删除
- 移除 `Dataset` 的 `unique_together`（见上，迁移记录）。无代码文件删除。
