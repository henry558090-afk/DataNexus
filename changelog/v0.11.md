# v0.11 版本变更说明

> 发布日期：2026-06-06　|　分支：**dev**（重构基线大改版 · 阶段 C 门户）　|　负责人：henry + AI
> 主题：**目录（部门/分类）+ 用户端数据门户浏览 + 下载（含可见性校验）**

## 一、本次变更

### 新增（feat）
- **后端 · 目录管理**（仅管理员）：`catalog` 序列化/视图——`DepartmentViewSet`、`CategoryViewSet`（?department 过滤）。
- **后端 · 用户端门户**（`apps/catalog/portal.py`，登录可访问，全部走 `can_view_dataset` 可见性）：
  - `GET /api/portal/tree/`：返回当前用户**可见的**部门→分类→数据集树（空分支裁掉，默认拒绝）。
  - `GET /api/portal/datasets/{id}/`：数据集详情（最新 + 历史成功版本），含可见性校验。
  - `GET /api/portal/executions/{id}/download/`：下载（含可见性校验，越权 403）。
- **前端 · 管理端**：
  - `views/admin/CatalogView.vue`：部门 / 分类两栏管理（增删、选部门看分类）。
  - `views/admin/DatasetView.vue`：数据集表单新增「归类」下拉（部门/分类），可空=未归类。
- **前端 · 用户端门户**：
  - `views/user/CatalogHome.vue`：欢迎 + 搜索 + 按部门/分类的**数据卡片墙**，点击进详情。
  - `views/user/DatasetDetail.vue`：最新版本（行数/大小/时间 + 下载）+ 历史版本下载（按 §8.4）。
  - `api/catalog.ts`、`api/portal.ts`（含带 Token 的 blob 下载）；新增用户路由 `/dataset/:id`。

### 修改 / 优化
- `health` 版本号 → v0.11。

### 删除
- 无。
