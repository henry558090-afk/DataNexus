# v0.12 版本变更说明

> 发布日期：2026-06-06　|　分支：**dev**（重构基线大改版 · 阶段 C 门户）　|　负责人：henry + AI
> 主题：**用户与权限（角色 / 部门成员 / 授权）**

## 一、本次变更

### 新增（feat）
- **后端 · 用户管理**（仅管理员）：`accounts` 序列化/视图——`UserViewSet`：
  - 创建/编辑用户、设管理角色（`is_assistant_admin`/`is_boss`）、启用停用、重置密码；
  - 密码 write_only；`is_superuser` 只读（**不能通过 API 提权为超管**）；
  - **辅助管理员看不到、也管不了超级管理员**（get_queryset 排除）。
- **后端 · 权限分配**（仅管理员）：`permission` 序列化/视图——
  - `MembershipViewSet`：部门成员（总监/主管/成员 + 看本部门全部），?user 过滤；
  - `GrantViewSet`：成员授权（个人或角色组 → 分类/数据集），?user 过滤。
- **前端 · 用户与权限页** `views/admin/PermissionView.vue`：
  - 用户表（角色标签/状态）+ 新建用户 + 角色设置（辅助管理员/老板/启用/重置密码）；
  - 「可见权限」对话框：部门成员增删（部门/角色/看全部）+ 按分类授权增删；
  - `api/permission.ts` 封装 users/memberships/grants。

### 修改 / 优化
- `health` 版本号 → v0.12。

### 删除
- 无。
