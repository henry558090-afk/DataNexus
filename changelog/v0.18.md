# v0.18 版本变更说明

> 发布日期：2026-06-07　|　分支：**dev**　|　负责人：henry + AI
> 主题：**管理端前端重做（适配文件夹模型）**

## 一、本次变更

### 新增 / 重做（feat）
- **API 层重写**：`api/catalog.ts`（文件夹/部门/文件夹授权）、`api/datafile.ts`（数据文件列表/下载，分页）、`api/dataset.ts`（目标文件夹/命名/保留/last_run）、`api/permission.ts`（用户/部门成员，去 grant）、`api/portal.ts`（文件夹树/文件/搜索/下载）、`api/stats.ts`（files）。
- **文件夹管理页** `views/admin/FolderView.vue`：目录树 + **新建/重命名/拖拽移动/删除** + **授权对话框**（文件夹 → 部门 或 个人）。
- **数据集页重做**：表单加 **目标文件夹 / 文件命名（前缀+日期格式）/ 定时 / 保留(份数/天数)**；列表显示文件夹、定时、最近结果；下载最新文件。
- **数据文件页** `views/admin/FileView.vue`：所有生成文件列表（分页）+ 下载（替代旧"执行记录"）。
- **用户与部门页** `views/admin/PermissionView.vue` 重做：用户 + 角色（辅助管理员/老板/启停/重置密码）+ **部门归属**（可多部门）+ 部门增删。
- **首页**统计改为 数据源/数据集/数据文件/今日生成 + 最近生成文件。
- 路由与菜单：`文件夹`/`数据文件`/`用户与部门` 替换旧 `目录管理`/`执行记录`/`用户与权限`。

### 删除（已告知）
- `views/admin/CatalogView.vue`、`views/admin/ExecutionView.vue`、`api/execution.ts`（被 FolderView/FileView/datafile.ts 取代）。
