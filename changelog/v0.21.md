# v0.21 版本变更说明

> 发布日期：2026-06-07　|　分支：**dev**　|　负责人：henry + AI
> 主题：**管理端功能拆分重组（Phase 2）+ 外壳套用新设计系统（Phase 3 起步）**

## 一、本次变更

### 管理端功能拆分（feat）—— 每个功能独立页，逻辑不再混在一起
- **目录管理**（`FolderView.vue` 重做）：文件夹目录树（建/重命名/拖拽移动/删）+ **把数据集挂载到文件夹**（设置数据集 target_folder，跑数文件落到此文件夹）。授权已移出。
- **权限管理**（`PermissionView.vue` 重做）：**选人/部门 → 勾选可见文件夹**（`check-strictly` 独立勾选，每勾一个即建/删一条 FolderShare；授权后端递归含子文件夹）。最细到人、一人多文件夹。
- **部门管理**（新 `DepartmentView.vue`）：部门 CRUD + 选部门管理其成员（加入/移出）。
- **人员管理**（新 `UserView.vue`）：用户 CRUD + 角色（辅助管理员/老板/启停/重置密码）。
- 旧"用户与部门"混合页拆解完毕；菜单按 **数据 / 权限 / 系统** 分组。

### 设计系统套用（Phase 3 起步，feat）
- `AdminLayout.vue` 重做：品牌/侧栏/头部套用新令牌（靛蓝、第二中性层、分组标签、active 弱底高亮），路由切换淡入。
- 新建/重做的四个管理页统一用新设计系统（卡片、列表项 hover/选中、空状态、rise-in）。
- 其余管理页（数据源/数据集/数据文件/审计/工作台）经令牌别名自动继承新浅色系。

### 后端（feat）
- `FolderShareViewSet` 支持 `?subject_user` / `?subject_department` 过滤，支撑"按主体看授权"。

## 二、验证
- 后端 `pytest` **70 全过**、`ruff`/`black` 清。前端 `build`(vue-tsc) + `eslint` 通过。

## 三、后续
- Phase 3 深化：数据源/数据集/数据文件/工作台 各页视觉细打磨（当前已继承新令牌，可用）。
- 等用户验收后按反馈优化。
