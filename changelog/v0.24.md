# v0.24 版本变更说明

> 发布日期：2026-06-11　|　分支：**dev**　|　负责人：henry + AI

迭代计划第 3 版。101 项测试通过（v0.23 的 94 + 新增 7），ruff 干净，前端 build 通过。

## 一、本次变更

### 在线预览 + 选列下载（feat，前后端完整）
- 后端 `GET /api/portal/files/{id}/preview/?limit=50`：流式读回 xlsx 前 N 行（`core.excel.read_xlsx`，read_only）。
- 下载支持 `?columns=a,b`：现场生成只含选中列的文件（`portal_download` 增强）。
- 用户门户每个文件新增"预览"按钮 → 弹窗看前 50 行 + 多选列 → 按选中列下载。

### 搜索增强（feat，后端 + API）
- `portal_search` 支持 `?from=&to=` 日期范围（配合原文件名关键词）。前端 `searchFiles(q, from, to)` 已就绪。

### 收藏文件夹（feat，后端 + API）
- 新模型 `Favorite(user, folder)`（迁移 `0003`）。
- `GET /api/portal/favorites/`（仅返回仍可见的）、`POST /api/portal/folders/{id}/favorite/` 切换收藏（带可见性校验）。

### 最近下载（feat，后端 + API）
- `GET /api/portal/recent-downloads/`：从审计日志取当前用户最近 10 条下载。

### 更新通知（feat，后端 + API）
- `GET /api/portal/updates/?since=ISO`：可见范围内 since 之后新增的成功文件 + 计数。前端可用 localStorage 存上次查看时间。
  - 注：query 传 ISO 时间须用 `Z` 结尾或正确 URL 编码（`+` 会被解码成空格），前端 `toISOString()` 天然为 `Z` 形式。
