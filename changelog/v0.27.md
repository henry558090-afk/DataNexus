# v0.27 版本变更说明

> 发布日期：2026-06-11　|　分支：**dev**　|　负责人：henry + AI
> 主题：**企业级 —— 列级脱敏 / 审计可视化 / 企业微信 SSO**

迭代计划第 6 版（收官）。123 项测试通过（v0.26 的 116 + 新增 7），ruff 干净，前端 build 通过。
**零额外依赖**：企微 SSO 的 HTTP 调用走标准库 urllib。

## 一、本次变更

### 列级脱敏（feat，前后端 + 配置）
- 新模型 `MaskingRule(dataset, column, strategy[full/partial])`（迁移 `0003`）。
- `apps/dataset/masking.py`：`mask_rows` 对命中列遮蔽（partial 保留首尾，full 全遮）。
- 门户**预览/下载**对**普通用户**自动脱敏；**管理员/超管/老板/数据集属主看原值**
  （`portal._should_see_raw`）。下载时若有脱敏规则则现生成脱敏文件。
- `MaskingRuleViewSet`（仅管理员）CRUD。

### 审计可视化（feat，后端 + API）
- `GET /api/audit-logs/stats/?days=N`：近 N 天总数、按动作分类、每日趋势、活跃用户 Top 10。

### 企业微信 SSO（feat，后端完整 + 配置 + API）
- `apps/accounts/wecom.py`：扫码授权 URL + code→userid（urllib，HTTP 集中可 mock）。
- `GET /api/auth/wecom/login/` 返回授权 URL；`/callback/` 用 code 换 userid →
  映射本地用户（`WECOM_AUTO_PROVISION` 可自动建号）→ 发 Token → 带 token 跳前端。
- 配置开关 `WECOM_ENABLED` 等（部署手册 §8.4）；未启用不影响账号密码登录。
